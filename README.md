BFS, DFS - 모든 정점을 방문하는 방법 두가지
------------------------------------------
   * 두 노드 사이의 최단 경로를 구할때 사용
   * 임의의 경로를 찾고 싶을때 사용

<img src="/img/bfsdfs.png" title="bfsdfs" alt="bfsdfs"></img><br/>

1. BFS - 너비 우선 탐색
   * **큐(queue)** 이용
   * 최단 경로 무조건 보장
   * 노드의 수가 많을수록  더 큰 저장공간 필요
2. DFS - 깊이 우선 탐색
   * **스택**, 또는 **재귀함수** 이용
   * 백트래킹을 해야하는 노드들만 저장해주면됨
   * 최단경로 보장X, 깊으면 시간 오래걸림
